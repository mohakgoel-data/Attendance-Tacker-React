Student Attendance Register
A modular React application for tracking, filtering, and monitoring student attendance records.

Features
Real-time Statistics: Calculates total students, present/absent counts, low attendance (<75%), and average attendance.

Smart Filtering & Sorting: Filter by status (Present/Absent), toggle low-attendance students, and sort by attendance percentage.

Interactive Data Table: Clickable rows that reveal a detailed panel for individual student records.

Modular Codebase: Clean architecture separating UI components, state management, and utility constants.
